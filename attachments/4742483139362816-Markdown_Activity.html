Sugar Labs

Sugar labs is a NGO a non-profit organisation cum community run software or open software whose purpose is for the benefit of not only software engineers but is also used by the younger generation as well as the older ones.

This open software is easy to use by everyone from our old grandpas to a young boy of 6 to 7 years who has some knowledge of using a computer and is lead to the proper way of using this software.

Various open software are tricky and require to learn various higher languages in terms of computer like C++,Java,etc. but this software doesn't need it and more over this software is available world wide.

Sugar labs has various programs arranged for interested youth,teachers and students like 'MONTEVIDEO YOUTH SUMMIT AND TURTLE ART DAY'
